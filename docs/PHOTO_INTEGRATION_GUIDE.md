# PHOTO_INTEGRATION_GUIDE

## 1. Homepage Photo Dimensions

- Hero portrait: 1600 x 2000 px preferred, vertical or slightly portrait.
- Evidence mosaic large tile: 1800 x 1400 px.
- Small mosaic tiles: minimum 1200 x 1200 px.
- Export WebP or optimized JPG, 75-85 quality.

## 2. Preferred Orientations

- Hero: portrait with Jimena visible in a real work context.
- Facilitation: horizontal or square, showing group interaction.
- Materials: close-up, hands, printed object, radio/audio setup, or workshop tool.
- Territory: horizontal, useful as a wide contextual tile.

## 3. Case-Study Photo Roles

- Hero image: strongest real evidence for the case.
- Action image: Jimena facilitating, coordinating, or accompanying.
- Context image: community, institution, field visit, meeting space.
- Object image: material, cartilla, guide, radio script, report.

## 4. Hero Image Requirements

- Must not imply CETM affiliation.
- Must have usage permission.
- Must show real context, not a staged stock-like portrait.
- Avoid low-resolution WhatsApp images unless no alternative exists.

## 5. Video Poster Requirements

- Use either a frame from the finished video or a verified portrait/workshop image.
- Do not use fake documentary imagery.
- Leave room for a play control and title overlay.

## 6. Compression / Export

- Use WebP where possible.
- Keep large hero images under 450 KB if quality allows.
- Keep small tiles under 250 KB.
- Name files clearly: `photo-workshop-cipca-2022-01.webp`.

## 7. Alt Text Guidance

- Describe what is visible, not the desired claim.
- Good: "Jimena facilita un taller con un grupo sentado en circulo."
- Avoid: "Jimena transformando comunidades con liderazgo ambiental."

## 8. Evidence Caption Guidance

- One to two short lines.
- Include organization/year/location only when verified.
- If metadata is incomplete, keep the caption generic or do not publish the photo.

## 9. Privacy Considerations

- Avoid publishing minors without explicit permission.
- Avoid sensitive participant details.
- Confirm permission for institutional photos.
- Blur or crop if a person should not be identifiable.

## 10. Avoid Duplicated Use

- Do not use the same photo as hero, case hero, and mosaic tile.
- Each image should serve one role.
- If only one strong image exists, use it prominently once and support it with object placeholders.
